# Parrot Device Dashboard

A React dashboard for managing and researching the Parrot Device — assign songs to buttons, manage your parrot's profile, and explore behavior analytics.

---

## Requirements

- Node.js v20 or higher
- npm (comes with Node.js)
- cppcheck, for Arduino static analysis

---

## Clone the Repository
git clone https://github.com/ludev-nl/2026-26-Puppets_for_Parrots.git
git checkout web-app-demo

# Quick Start

# Windows

npm install 
npm run dev

# Open the app in your browser on http://localhost:8080


# macOS / Linux

bash start.sh 

# Then open http://localhost:8080


# run unit tests:
npm test
